# Обработка статей через DeepSeek API

Скрипт `process_with_deepseek.py` отправляет текст каждой статьи из `articles_markdown/` в DeepSeek API и записывает ответ в заголовок H1 файла.

## Запуск

```bash
pip install openai
python3 process_with_deepseek.py <DEEPSEEK_API_KEY>
```

При запуске скрипт покажет список `.txt` файлов в корне проекта и предложит выбрать файл системного промпта (по номеру или по пути).

## Логика работы

1. Принимает API ключ DeepSeek как аргумент командной строки
2. Запрашивает у пользователя файл системного промпта
3. Последовательно обрабатывает каждый `.md` файл из `articles_markdown/`:
   - Извлекает тело статьи (всё после разделителя `---`)
   - Удаляет изображения из текста
   - Отправляет очищенный текст в DeepSeek API
   - Заменяет H1 заголовок (`# ...`) на ответ API
4. Выводит итоговый отчёт со старыми и новыми заголовками

## Файл системного промпта

Текстовый файл, содержимое которого передаётся в параметре `system` при каждом запросе к API. Определяет задачу для модели (например, генерация заголовков, суммаризация и т.д.).
