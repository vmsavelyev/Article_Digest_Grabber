# Обработка статей через DeepSeek API

Скрипт `process_with_deepseek.py` отправляет текст каждой статьи из `articles_markdown/` в DeepSeek API и записывает ответ в заголовок H1 файла.

## Запуск

```bash
pip install openai
python3 process_with_deepseek.py <DEEPSEEK_API_KEY>
```

## Логика работы

1. Принимает API ключ DeepSeek как аргумент командной строки
2. Показывает список `.txt` файлов в корне проекта и запрашивает выбор файла системного промпта (по номеру или по пути)
3. Запрашивает количество параллельных запросов (по умолчанию 5)
4. Параллельно обрабатывает `.md` файлы из `articles_markdown/`:
   - Извлекает тело статьи (всё после разделителя `---`)
   - Удаляет изображения из текста
   - Отправляет очищенный текст в DeepSeek API
   - Заменяет H1 заголовок (`# ...`) на ответ API
5. Выводит итоговый отчёт со старыми и новыми заголовками

## Файл системного промпта

Любой `.txt` файл, содержимое которого передаётся в параметре `system` при каждом запросе к API. Определяет задачу для модели (например, генерация заголовков, суммаризация и т.д.).
